TCL_READ - load a Tcl file into a cell array of lines.
  LINES = TCL_READ(TCLFILE)
  PURPOSE : To load a Tcl file into a cell array of lines.
                      This function removes comments and spaces at the
                      biginning of the line automatically.
  USAGE :    lines = tcl_read([tclfile])
