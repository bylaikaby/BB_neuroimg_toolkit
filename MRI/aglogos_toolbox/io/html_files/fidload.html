FIDLOAD - Load Paravision fid files
usage: Ofid = FIDLOAD(Ses,ExpNo)
Ses: session name or structure
ExpNo: Experiment number
Returns: tcFid structure with (e.g.)
  tcFid =
          session: 'phantomsc1'
          grpname: 'bigcoil1'
              ExpNo: 1
                  dir: [1x1 struct]
                  dsp: [1x1 struct]
                  usr: [1x1 struct]
                  dat: [4-D int32]
                    ds: [0.7500 0.7500 2]
                    dx: 0.2500

    VERSION : 08.11.04 YM  first release

    See also PVRDFID
