TXT_READ - load a Tcl file into a cell array of lines.
  LINES = TXT_READ(TXTFILE,KEEPBLANK)
  PURPOSE : To load a Tcl file into a cell array of lines.
                      This function removes comments and spaces at the
                      biginning of the line automatically.
  USAGE :    lines = txt_read(txtfile)
  VERSION : 0.90  08/12/03  YM, modified from load_tclfile.m
