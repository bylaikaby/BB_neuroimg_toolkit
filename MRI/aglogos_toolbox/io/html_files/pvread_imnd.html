PVREAD_IMND - read "imnd" and returns contents as a structure.
    IMND = PVREAD_IMND(IMNDFILE)
    IMND = PVREAD_IMND(2DSEQFILE)
    IMND = PVREAD_IMND(SESSION,EXPNO)  reads ParaVision's "imnd" and returns
    its contens as a structre IMND.

    VERSION :
        0.90 : 13.06.05 YM  pre-release
        0.91 : 14.06.05 YM  checked both 'mdeft' and 'epi'.
        0.92 : 27.02.07 YM  supports also 2dseq as the first argument

    See also GETPVPARS, PVREAD_ACQP
