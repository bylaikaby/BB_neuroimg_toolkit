Preprocessing for Neurophysiology Data


>> Session = 'a98nmX';
>> goto(Session)
>> sesdumppar(Sessin)                  % saving experiment parameters into "SesPar.mat"
>> sesgetcln(Session)                  % creating Cln signal (decimated full-band signal)
>> sesclnspc(Session)                  % creating ClnSpc, spectgram of Cln
>> sesgetlfpmuaflt(Session)            % computing Lfp/Gamma/Mua by band-pass filtering from Cln
>> sesgetspk(Session)                  % extracting spikes from Cln, simply by threshoulding data
>>
>> sigload(Session,1,'Mua','ClnSpc')   % loading extracted Mua and ClnSpc
>>
>> showchan(Mua)                       % plotting Mua
>> show(ClnSpc)                        % plotting ClnSpc

Frequency bands for Gamma/Lfp/Mua/LfpLHM can be defined in the description file like
ANAP.bands.Lfp          = [  0  500];    % entire "lfp" range
ANAP.bands.GammaL       = [ 24   35];    % gamm "lfp" range (unrectified)
ANAP.bands.GammaM       = [ 35   80];    % delta-theta (envelop)
ANAP.bands.GammaH       = [ 80  100];    % alpha-beta (envelop)
ANAP.bands.Mid          = [100  400];    % gamma (envelop)
ANAP.bands.Mua          = [400 3000];    % spiking
ANAP.bands.lfpcutoff    = 10;            % Low pass filter after rectification
ANAP.bands.muacutoff    = 100;           % Low pass filter after rectification
ANAP.bands.samprate     = 250;           % Resample at 250Hz

% for backward compatibility
anap.bands.LfpL         = [ 1 12];        % delta-theta (envelop)
anap.bands.LfpM         = [12 24];        % alpha-beta (envelop)
anap.bands.LfpH         = [24 90];        % gamma (envelop)
anap.bands.Gamma        = [24 90];        % gamm "lfp" range (unrectified)

Visualization of Neurophysiology Data

nview can display continuous signals such as Cln, blp etc.  spkview can display spike timings in Spkt signal.

EXAMPLE :
>> nview(Session,GrpName,'blp');
>> spkview(Session,GrpName);

Receptive Field Mapping by movie stimuli

To map receptive fields (RFs) of recording sites by movie clips, two additional signals (1. swap signal from OpenGL card, 2. photodiode signal placed over  small flickering area (white rectangle) near the right edge of the movie clip) must be recored at the same time with neuronal signals.  When generating Cln structure, vdecmain or vclnmain will be called automatically from sesgetcln function and it attaches movie frame information into Cln structure as Cln.movie.  sesgetrf averages movie frames around threshoulded neural data and generates VMua3 or VSdf3 structure (prefix of "V" and postfix of threshould).  The extracted RF data can be visualized with showrf function.  Parameters for RF extraction can be set as ANAP.revcorr.xxx like following.  TOFFSET of  -50 to 0ms will show some RF structure for V1 recording.
ANAP.revcor.TOFFSET    = [0];  % Time onset in seconds for movie averaging
ANAP.revcor.LFP_THR    = [3];  % threshould for LFP
ANAP.revcor.MUA_THR    = [3];  % threshould for MUA
ANAP.revcor.Frame      = 1;                    % For display purpose
ANAP.revcor.BadRFChan  = [2 14 15];            % For display purpose
ANAP.revcor.MovPos     = [-1.8 -2.9 14 10.5];  % For display purpose

EXAMPLE :
>> sesgetcln(Session,GrpName);
>> sesgetblp(Session,GrpName);
>> seslfpmuaflt(Session,GrpName);
>> sesgetrf(Session,GrpName);
>> sesgrpmake(Session,GrpName,{'VMua3','VLfp3'});
>> showrf(Session,GrpName);

Functions for Signal Extraction

sesclnspc (Ses, Exps/GrpName) Generate ClnSpc, spectrograms for denoised signal Cln
sesgetlfpmua (Ses, Exps/GrpName) Generate pMua/pLfp etc, average spectrogram bands from ClnSpc
sesgetlfpmuaflt (Ses, Exps/GrpName) Generate Gamma/Lfp/Mua/LfpLHM by filter, rectify and decimate
sesgetblp (Ses, Exps/GrpName) Separate the Cln signal into freqeuncy bands, blp
sesgetspk (Ses, Exps/GrpName) Extract spike times and create Spike Density Functions, Spkt/Sdf
sesspktrigavr (Ses, Exps/GrpName) Computes spike triggered averages of signal, SpktCln etc
sesgetrf (Ses, Exps/GrpName) Compute RF of the cells by means of reverse correlation.

sigspc (Sig,  WT, dT, NFFT, WinFct) Make spectrogram from the given signal
getlfpmua (Ses, ExpNo) Extract power signals from ClnSpc (LfpPow,MuaPow)
getbandsflt (Ses, ExpNo, SigNames) Extract Gamma/Lfp/Mua/LfpLHM by bandpass filtering
expgetblp (Ses, ExpNo) Separate the Cln signal into freqeuncy bands for SesName/ExpNo
getspk (Ses, ExpNo) Extracts spikes and SDFs from Cln.dat
sigspktrigavr (Spkt, Sig, MaxLagsInSec) Computes spike-triggered average of SIG.
getrf (Ses, ExpNo) Compute RF of the recording sites calling siggetrf() function
siggetrf (Sig,Threshold,...) Compute site-RF structure by means of reverse correlation.
