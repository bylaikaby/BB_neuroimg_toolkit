MNVIEW - displays anatomical images
    MNVIEW(SESSION,GRPNAME) displays anatomical images for SESSION/GRPNAME.
    MNVIEW(SESSION,ANANAME,INDEX) displays anatomical images for ANANAME/INDEX.
    MNVIEW(SIG) displays anatomical data "SIG".

    MNVIEW(SESSION,GRPNAME,PERMUTE_VEC)
    MNVIEW(SESSION,ANANAME,INDEX,PERMUTE_VEC)
    MNVIEW(SIG,PERMUTE_VEC) performs permutation of image data.


    NOTES :
        This function assumes .dat as (x,y,z) as default.

    VERSION :
        04.07.05 YM    pre-relase
        05.07.05 YM    supports permutation/gamma/crosshair.
